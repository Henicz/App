Instrukcja obsługi


Informacje ogólne
Aplikacja CodeStructure umożliwia proste zarządzanie stanem chmury publicznej Microsoft Azure z poziomu kodu infrastrukturalnego. Narzędzie ułatwia to tworzenie, edycję oraz dokumentację zasobów utworzonych w chmurze.
Konfiguracja

  1. Utwórz Service Principal w Microsoft Azure AD.
  2. Nadaj odpowiednie uprawnienia dla utworzonego Service Principal na poziomie Subskrypcji Microsoft Azure - zalecana rola to 'Contributor'.
  3. Zbierz poniższe dane utworzonego Service Principal:
    • Object ID
    • Secret - należy go wygenerować
  4. W zakładce Autoryzacja utwórz nowy obiekt i zapisz zebrane dane.
  5. Utwórz Service Principal w Microsoft Azure AD.

Funkcjonalności
  • Tworzenie nowych zasobów
  • Nieinwazyjna weryfikacja zmian
  • Edycja istniejących zasobów
  • Odświeżenie zapisanego stanu zasobu
  • Eksport pliku konfiguracyjnego zasobu
